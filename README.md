## Features

- Liquid glass morphism design
- Custom cursor effects
- Hero section with stunning visuals
- Features showcase
- Logo marquee (client logos)
- Pricing section
- Smooth scroll experience
- Responsive design
- Dark mode support

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19, Tailwind CSS 4
- **3D Graphics:** OGL (OpenGL-like library)
- **Animations:** CSS Transitions & Transforms
- **Typography:** Geist
- **Language:** TypeScript

## Getting Started

### Prerequisites

- Node.js 18+
- pnpm (recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/raheem-dotgit/modern-agency-liquid-glass.git

# Navigate to the project directory
cd modern-agency-liquid-glass

# Install dependencies
pnpm install

# Start the development server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Customization

1. Update agency name and branding in header
2. Add your services in the features section
3. Add client logos in the marquee
4. Update pricing plans
5. Modify glass effect colors and blur

## Glass Morphism

The liquid glass effect is achieved using:
- Backdrop blur filters
- Semi-transparent backgrounds
- Subtle borders and shadows
- CSS transforms on hover
